# 🍽️ Food Menu Web App

## 📌 Description
เว็บแสดงเมนูอาหาร พร้อมระบบ Admin สำหรับเพิ่ม แก้ไข ลบ และอัปโหลดรูป

## 🚀 Tech Stack
- Frontend: Vue 3 + Tailwind CSS
- Backend: Django
- Database: SQLite

## ✨ Features
- 🔐 Admin Login
- ➕ เพิ่ม / แก้ไข / ลบเมนู
- 🖼️ อัปโหลดรูปภาพ
- 🔍 ค้นหาเมนู
- 📄 หน้า Menu Detail

## ▶️ วิธีรันโปรเจค

```bash
python manage.py runserver

### Frontend
